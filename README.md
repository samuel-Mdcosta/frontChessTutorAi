## What it does

This is the web interface for the [ChessTutor AI backend](https://github.com/samuel-Mdcosta/chessTutor-AI). Players and coaches use it to:

- **Upload PGN files** (up to 20 games at once) and submit them for analysis
- **Visualize move-by-move diagnostics** — blunders, mistakes, centipawn loss, best alternatives
- **Read AI coaching reports** generated by Gemini, grounded in real Stockfish engine data
- **Browse game history** with per-game performance summaries
- **Manage accounts** with authentication (login / register)

## Application screens

| Route | Page | Description |
|---|---|---|
| `/` | Login | Authentication entry point |
| `/create-account` | CreateAccount | New user registration |
| `/analize` | Analyze | PGN upload and analysis submission |
| `/coach` | Coach | AI coaching feedback view |
| `/games` | Games | Game history list |

## Architecture

```
User
  │
  ▼
┌─────────────────────────────────────┐
│           React 19 SPA              │
│                                     │
│  App.jsx                            │
│  └── React Router 7 (client-side)  │
│      ├── /           → Login        │
│      ├── /create-account            │
│      ├── /analize    → PGN upload   │
│      ├── /coach      → AI feedback  │
│      └── /games      → History      │
│                                     │
│  Axios (HTTP client)                │
└──────────────┬──────────────────────┘
               │
               ▼
   ┌───────────────────────┐
   │   ChessTutor AI API   │
   │  FastAPI + Stockfish  │
   │  + Gemini (backend)   │
   └───────────────────────┘
```

**API endpoints consumed:**

| Environment | Base URL |
|---|---|
| Development | `http://localhost:8000` |
| Production | `https://chesstutor-ai.onrender.com` |

## Getting started

### Prerequisites

- Node.js 18+
- The [ChessTutor AI backend](https://github.com/samuel-Mdcosta/chessTutor-AI) running locally (or use the production URL)

### Installation

```bash
git clone https://github.com/samuel-Mdcosta/frontChessTutorAi.git
cd frontChessTutorAi
npm install
```

### Running in development

```bash
npm run dev
```

The app will start at `http://localhost:5173` and proxy API requests to the backend at `http://localhost:8000`.

### Building for production

```bash
npm run build
npm run preview  # preview the production build locally
```

## Deployment

This frontend is deployed on **Vercel** with SPA routing handled via `vercel.json` (rewrites all paths to `index.html` for client-side routing to work correctly on refresh).

The backend API is deployed on **Render** at `https://chesstutor-ai.onrender.com`.
